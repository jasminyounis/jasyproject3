Project 3: Alice

About the project

The inspiration behind this project was the song Alice from Lady Gaga's recent album Chromatica. Chromatic is an album that touches on Gaga's journey with mental health. The land of Chromatica is similar to Wonderland filled with twists and turns, but overall there is a path that you end up finding yourself.

Artist statement: Listening to the song,Alice,reminds me of feeling like Alice in Wonderland, that feeling of being lost and alone but ultimately taking the chance to find yourself. While going through the Three.js website, I found the teapot, and I thought it would be interesting to create a tea set on table and watch how the reflections of the environment interact with the glass tea set or how using javascript to create glitch features on the tea set could represent the imperfections.
